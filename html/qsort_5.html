qset_mbqs R Documentation

MBQS Q-set criteria scores and derived scales

Description

A data frame containing the Maternal Behaviour Q-set (mbqs; version 3.1). In this dataset, column names ending in "_c" refer to criteria sorts' scores, while column names ending in "_s" refer to scales' scores.

Usage

qset_mbqs

Format

MBQS dataset with 90 rows (items) and 4 columns.

item

Item number from 1 to 90.

qset

mbqs. Maternal Behaviour Q-set (version 3.1) (Pederson et al., 1999).

sens_s

Sensibility criterion scores (retrieved from Pederson et al., 1999).

description

Item description (retrieved from Pederson et al., 1999).

References

Pederson, D. R., Moran, G., & Bento, S. (1999). Maternal Behaviour Q-sort (version 3.1). London, ON: Psychology Department, Western University.
