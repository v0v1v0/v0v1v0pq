qrencode_png R Documentation

Return a QR encoded string as a base 64 encoded inline png

Description

Return a QR encoded string as a base 64 encoded inline png

Usage

qrencode_png(to_encode)

Arguments

to_encode

the data to encode

Note

data:image/png;base64, is prepended to the encoded png

Examples

cat(qrencode_png("http://rud.is/b"))
