Inverse Weibull function

Description

Inverse Weibull function

Usage

inv_weibull_fun(q, p)

Arguments

q

Vector of probabilities.
p

Vector of parameters p = c(alpha, beta).

Value

x at each probability.

See Also

weibull_fun

Examples

yseq <- seq(0, 1, .01)
xseq <- inv_weibull_fun(yseq, c(2, 4))
curve <- data.frame(x = xseq, y = yseq)
ggplot(curve, aes(x = x, y = y)) + geom_line()
