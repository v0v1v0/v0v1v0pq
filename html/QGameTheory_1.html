Bell States

Description

The function builds one of the four Bell states, according to the input qubits

Usage

Bell(qubit1, qubit2)

Arguments

qubit1

1st input qubit
qubit2

2nd input qubit

Value

One of the Bell states as a vector depending on the input qubits.

Examples

init()
Bell(Q$Q0, Q$Q0)
Bell(Q$Q0, Q$Q1)
Bell(Q$Q1, Q$Q0)
Bell(Q$Q1, Q$Q1)
