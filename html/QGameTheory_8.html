levi_civita R Documentation

Levi-Civita symbol

Description

This function computes the Levi-Civita symbol depending on the permutations of the three inputs, lying in 0 to 2

Usage

levi_civita(i, j, k)

Arguments

i

an integer 0, 1 or 2
j

an integer 0, 1 or 2
k

an integer 0, 1 or 2

Value

0, 1 or -1 after computing the Levi-Civita symbol depending on the permutations of the three inputs 0, 1 and 2

References

https://en.wikipedia.org/wiki/Levi-Civita_symbol

Examples

init()
levi_civita(0, 2, 1)
levi_civita(1, 2, 0)
levi_civita(1, 2, 1)
