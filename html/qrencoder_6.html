qrencode_raster R Documentation

Return a QR encoded string as a raster object

Description

Return a QR encoded string as a raster object

Usage

qrencode_raster(to_encode)

Arguments

to_encode

the data to encode

Examples

library(raster)
old_mar <- par()$mar
par(mar=c(0,0,0,0))
image(qrencode_raster("http://rud.is/b"), asp=1, col=c("white", "black"),
                                          axes=FALSE, xlab="", ylab="")
par(mar=old_mar)
