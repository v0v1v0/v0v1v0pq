Description

Rename a data frame column

Usage

rename.variable(df, old, new)

Arguments

df

data frame

old name
new

new name

Value

A data frame with the column named "old" renamed as "new"

Examples

data(iris)
str(iris)
iris <- rename.variable(iris, "Species", "especes")
