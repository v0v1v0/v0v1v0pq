check_identical R Documentation

Check whether a vector, x, has all its entries equal to its first entry

Description

Check whether a vector, x, has all its entries equal to its first entry

Usage

check_identical(x)

Arguments

x

a vector

Value

a logical indicating whether all vector entries are the same

Examples

x <- 1:5
check_identical(x)
y <- rep(1, 5)
check_identical(y)
