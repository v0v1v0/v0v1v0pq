mintermMatrix R Documentation

Create Minterm Matrices

Description

This function creates minterm and implicant matrices. It is mainly used for internal and demonstration purposes.

Usage

mintermMatrix(noflevels, logical = FALSE)

Arguments

noflevels

The number of levels for each exogenous factor.
logical

Logical, return the matrix in logical values (only bivalent data).

Details

Minterm matrices contain all unique and complete conjunctions that can be formed from all levels of k factors (Dusa and Thiem 2015). The total number of minterms d is given by d = \prod_{j = 1}^{k}{p_{j}}, where p_{j} is the number of levels for exogenous factor j and k is the total number of exogenous factors. A minterm matrix is an essential part of a truth table.
