derivQspray R Documentation

Partial derivative

Description

Partial derivative of a qspray polynomial.

Usage

derivQspray(qspray, i, derivative = 1)

Arguments

qspray

object of class qspray
i

integer, the dimension to differentiate with respect to, e.g. 1 to differentiate with respect to x
derivative

integer, how many times to differentiate

Value

A qspray object.

Examples

library(qspray)
x <- qlone(1)
y <- qlone(2)
qspray <- 2*x  + 3*x*y
derivQspray(qspray, 2) # derivative w.r.t. y
