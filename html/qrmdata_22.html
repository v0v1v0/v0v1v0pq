commodities R Documentation

Commodity Data

Description

Data sets containing commodities.

Usage

data("OIL_Brent")
data("GOLD")

Format

xts objects containing the Brent Crude price in USD per barrel (for OIL_Brent) and the World Gold Council gold price in USD per troy ounce (for GOLD).

Author(s)

Marius Hofert

Source

The data was obtained from Federal Reserve Economic Data (FRED) via Quandl on 2016-01-03 with the function get_data() from qrmtools.

Examples

data("OIL_Brent")
data("GOLD")
