col_count R Documentation

Number of columns of a vector/matrix

Description

This function counts the number of columns of a vector or a matrix

Usage

col_count(M)

Arguments

M

A vector/matrix

Value

An integer that gives the number of columns in a vector or a matrix.

Examples

init()
col_count(Q$Q11)
col_count(Q$lambda4)
col_count(Q$I2)
