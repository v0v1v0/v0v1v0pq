get.Qmat R Documentation

get.Qmat

Description

Provide matrix Q derived from a matrix Q* of Helmert contrasts:

Q = Q^* (Q^{*T} Q^*)^{-1/2}

See Eq. A6 in Evin et al., 2019.

Usage

get.Qmat(p)

Arguments

p

integer

Value

matrix

p x p matrix

Author(s)

Guillaume Evin
