depart R Documentation

get departure times from queue_list object

Description

get departure times from queue_list object

Usage

depart(x)

Arguments

x

an queue_list object

Value

departure times

Examples

arrivals <- cumsum(rexp(10))
service <- rexp(10)
queue_obj <- queue_step(arrivals, service)

depart(queue_obj)
queue_obj$departures_df$departures
