CSWAP R Documentation

The CSWAP or Fredkin gate

Description

The CSWAP or Fredkin gate

Usage

CSWAP(bits = c(1, 2, 3))

fredkin(bits = c(1, 2, 3))

Arguments

bits

integer vector of length two, the first bit being the control and the second the target bit.

Value

An S4 class 'cswapgate' object is returned
