B_erlang R Documentation

Returns the probability that all servers are busy

Description

Returns the probability that all servers are busy

Usage

  B_erlang(c=1, u=0)

Arguments

c

numbers of servers
u

lambda/mu, that is, ratio of rate of arrivals and rate of service

Details

Returns the probability that all servers are busy
