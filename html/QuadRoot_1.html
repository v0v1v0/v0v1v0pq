QuadRoot R Documentation

Quadratic Root for any Quadratic Equation

Description

It is used to find simple quadratic roots from any quadratic equation. This package follows the Sridharacharya method to solve a quadratic equation.

Usage

QuadRoot(inputdata)

Arguments

inputdata

Coefficient vector of quadratic equation

Value

It returns quadratic roots of the equation

Author(s)

Pankaj Das

Examples

dataset=c(2,5,-3)
QuadRoot(dataset)
