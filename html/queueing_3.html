CheckInput R Documentation

Generic S3 method to check the params of a queueing model (or network)

Description

Generic S3 method to check the params of a queueing model (or network)

Usage

  CheckInput(x, ...)

Arguments

x

a object of class i_MM1, i_MMC, i_MM1K, i_MMCK, i_MM1KK, i_MMCKK, i_MMCC, i_MMCKM, i_MMInfKK, i_MMInf, i_OJN
...

aditional arguments
