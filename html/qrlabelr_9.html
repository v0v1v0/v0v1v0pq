run_app R Documentation

Launch 'EasyQrlabelr'.

Description

An interface function to launch the 'EasyQrlabelr' 'shiny' app.

Usage

run_app()

Value

No return value, called for side effects. Opens the 'EasyQrlabelr' 'shiny' app in your default browser.

Examples

library(qrlabelr)
if(interactive())
qrlabelr::run_app()
