Quandl.api_key R Documentation

Query or set Quandl API key

Description

Query or set Quandl API key

Usage

Quandl.api_key(api_key)

Arguments

api_key

Optionally passed parameter to set Quandl api_key.

Value

Returns invisibly the currently set api_key.

Examples

## Not run: 
Quandl.api_key('foobar')

## End(Not run)
