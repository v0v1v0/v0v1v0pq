Set up col, pattern and group for plotting.

Usage

color_patterns_get(scan1ggdata, col, palette = NULL)

Arguments

scan1ggdata

data frame to be used for plotting

palette

for colors (default NULL uses "Dark2" from RColorBrewer package)
