merge_dictionary_values R Documentation

Internal function to merge values of duplicated keys

Description

Internal function to merge values of duplicated keys

Usage

merge_dictionary_values(dict)

Arguments

dict

a dictionary object

Examples

dict <- list("A" = list(AA = list("aaaaa"), "a"),
             "B" = list("b"),
             "C" = list("c"),
             "A" = list("aa"))
quanteda:::merge_dictionary_values(dict)
