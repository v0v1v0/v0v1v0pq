Description

Angular coordinates to X, Y, Z.

Usage

llh2xyz(lonlatheight, rad = 6378137, exag = 1)

Arguments

lonlatheight

matrix or data.frame of lon,lat,height values
rad

radius of sphere
exag

exaggeration to apply to height values (added to radius)

Value

matrix
