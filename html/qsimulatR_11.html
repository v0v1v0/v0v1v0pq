CCNOT R Documentation

The CCNOT or toffoli gate

Description

The CCNOT or toffoli gate

Usage

CCNOT(bits = c(1, 2, 3))

toffoli(bits = c(1, 2, 3))

Arguments

bits

integer vector of length two, the first bit being the control and the second the target bit.

Value

An S4 class 'ccnotgate' object is returned
