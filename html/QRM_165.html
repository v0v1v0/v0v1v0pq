spdata.raw R Documentation

Standard and Poors Default Data

Description

The spdata.raw timeSeries contains default data for A, BBB, BB, B and C-rated companies for the years 1981 to 2000.

Usage

data(spdata.raw)
data(spdata.raw.df)

Source

Standard & Poors Credit Monitor

Examples

data(spdata.raw)
head(spdata.raw)
