%s% R Documentation

Operators

Description

A set of helpful operators to make writing and basic data analysis easier.

Usage

e1 %s% e2

Arguments

e1

a character string
e2

a character string

Examples


# base R
paste0("A longer string ", "can be ", "built")

# with the %s% operator
"A longer string " %s% "can be " %s% "built"
