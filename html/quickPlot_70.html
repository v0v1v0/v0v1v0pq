Fortify

Description

Convert an arbitrary object to a data.frame-like object.

Usage

ffortify(
  x,
  matchFortify = TRUE,
  simple = FALSE,
  maxNumPolygons = getOption("quickPlot.maxNumPolygons", 3000),
  verbose = getOption("quickPlot.verbose")
)

Note

This only deals with SpatialPolygons objects.
