Description

Returns the mean time spend in queue in the M/M/Infinite queueing model

Usage

  ## S3 method for class 'o_MMInf'
Wq(x, ...)

Arguments

x

a object of class o_MMInf
...

aditional arguments

Details

Returns the mean time spend in queue in the M/M/Infinite queueing model
